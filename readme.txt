Ozpost MultiQuote Shipping module. 

== Description ==

Supplies real time quotes from Australia Post, TNT Australia, SmartSend, Couriers Please, and Others... 

== Installation ==

1. Upload the entire 'woocommerce-shipping-ozpost' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
